GastroChef - The Lost Menu (Scaffold)

Quick scaffold for the Fullstack project (Express + Mongo / Vite + React).

Run server:

```bash
cd server
npm install
# configure .env
node seed.js   # optional seed
npm start
```

Run client:

```bash
cd client
npm install
npm run dev
```

This scaffold provides a minimal backend with auth, lab and marketplace routes and a React front-end with a 3x3 craft grid.
